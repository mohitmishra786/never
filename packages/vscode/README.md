# Never VS Code Extension

*"A constraint engine for your AI, right inside your editor."*

This extension integrates **Never** directly into VS Code, ensuring your AI assistant (Cursor, Copilot, etc.) respects your project's constraints.

---

## Features

*   **Auto-Sync:** Austomatically syncs rules when you save `config.yaml`.
*   **Linting:** Highlights violations of "Never" rules in your editor.
*   **Commands:** Run `Never: Sync Rules` and `Never: Scan Project` from the command palette.

## Setup

1.  Install the extension.
2.  Run `Never: Init` to create a configuration file.
3.  Edit `.never/config.yaml` to enable rule sets.
